Boundarizr

Boundarizr checks the styles of your elements to see if they can act as layout boundaries. The idea is to allow you to quickly check if you have more control over the degree of layout invalidations in your projects.

The box to the right is inside a div that satisfies the criteria for a layout boundary, and Boundarizr marks it with a blue box. If you were to check Timeline in Chrome DevTools you will see layout operations scoped to #innercontainer.

For more info read Wilson Page's intro to Layout Boundaries.
